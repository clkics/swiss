# 시시태태공장장 변경 이력

버전 규칙: 게임 앱은 v메이저.마이너.패치 (기능 추가 = 마이너, 수치·버그 수정 = 패치), 카드 DB는 별도 버전(v4.0 등).
게임 파일: index.html (= ganjang_game.html, 배포용) · 카드 DB: card_db_v4.json

## 게임 v1.1.0 — 카드 DB v4.0 (2026-07-18)

- 게임 제목 변경: 간장공장공장장 → **시시태태공장장**
- **7세 연령 추가**: 5세와 동일하게 TTS 읽어주기 + 따라 말하기, 단어 난이도 상향 (의성어·의태어 4자 + 쉬운 복합어 5자, 일반 50 + 함정 10)
- 판 최대 장수 통일: 연령별 8/10/12장 → **전 연령 15장**
- 카드 배치: 자동 축소 그리드 → **한 줄 5장 고정** (15장 = 3줄)
- 5세·7세는 카드 이미지 크게 표시 유지
- 설정 화면·게임 화면에 버전 표시 추가

## 게임 v1.0.0 — 카드 DB v3.1 (2026-07-18)

- 누적 읽기 방식 도입: 뒤집힌 카드가 판에 쌓이고, 차례마다 1번부터 전부 읽기
- 라운드 규칙: 실패 시 판 카드 전부 벌점, 판 정리 / 판 상한 채우고 성공 시 라운드 클리어 / 벌점 최소가 승리
- 제한시간 = 장당 시간 × 판 카드 수 (성인 2.5초, 15세 3.5초, 10세 5초 / 전문가 ×0.7), 전체 시간만 표시
- 함정카드 점 규칙: 이동 블록 전체 글자에 점 — 앞으로 온 블록 파란점, 뒤로 간 블록 빨간점
- 함정카드 = 어순 섞기 (예: 검은색반팔티 → 반팔색검은티)
- 5세 TTS 순차 읽어주기 + 현재 카드 하이라이트, 다시 듣기 버튼
- 특수카드 4종×2 (방향전환·한번더·패스·다같이), 클릭/Enter 진행, O/X 판정

## 카드 DB 이력

- **v4.0**: 7세 추가 (일반 50 + 함정 10), boardCap 전 연령 15
- **v3.1**: 누적 읽기 설정(perCardSec, boardCap) 추가, repeat 폐기
- **v3.0**: 함정카드 점 방식 변경 (blueDots/redDots)
- **v2.0**: 일반카드를 실제 발음 어려운 단어로 전면 교체(인터넷 경험담 조사 기반), 함정카드 어순 섞기 방식 도입, 전 카드 이모지 이미지
- **v1.0**: 최초 설계 (연령 4종 × 일반 50/함정 10, 특수 4종×2)
